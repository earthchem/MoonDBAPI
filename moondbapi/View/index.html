MoonDB REST APIs

The fastest way to pull data out from MoonDB

REST API Reference

MoonDB provides extensive access to instances through a set of RESTful APIs. Below you will find a list of the available endpoints with the latest information. For more information about a particular endpoint, click on "REST APIs" on the top navigation bar to view a description of the endpoint, applicable query parameters, example queries. All APIs access is over HTTP, and accessed from the http://api.moondb.org. All data is received as JSON.

Specimens APIs

URL Format Description Parameters Example Query
/specimen/{specimenCode} Get one specimen with basic meta data by specimen code specimenCode http://api.moondb.org/specimen/12023,155_7A
/specimenlist/mission/{missionName} Get specimens with basic meta data by mission name missionName http://api.moondb.org/specimenlist/mission/Apollo 11
/specimenlist/landmark/{landmarkName} Get specimens with basic meta data by landmark name landmarkName http://api.moondb.org/specimenlist/landmark/SPUR CRATER
/specimenlist/specimentype/{specimenType} Get specimens with basic meta data by specimen type specimenType http://api.moondb.org/specimenlist/specimentype/Regolith Breccia
/specimenlist/samplingtechnique/{samplingTechnique} Get specimens with basic meta data by sampling technique samplingTechnique http://api.moondb.org/specimenlist/samplingtechnique/Rake

Analysis Results APIs

URL Format Description Parameters Example Query
/data/specimen/{specimenCode} Get analysis results by specimen code specimenCode http://api.moondb.org/data/specimen/12023,155_7A
/data/analyzedMaterial/{analyzedMaterial} Get analysis results by analyzed material analyzedMaterial http://api.moondb.org/data/analyzedmaterial/LEACH
/data/citation/{citationCode} Get analysis results by citation citationCode http://api.moondb.org/data/citation/AGRELL, 1970
/data/{filter} Get analysis results by combined filter conditions including mission, landmark, specimenType, samplingTechnique, analyzedMaterial, analyte and analysisMethod {filter} is JSON object like {"mission":[],"landmark":[],"specimenType":[],"samplingTechnique":[],"analyzedMaterial":[],"analyte":[],"analysisMethod":[]} http://api.moondb.org/data/{"mission":[],"landmark":[],"specimenType":["Breccia"],"samplingTechnique":[],"analyzedMaterial":["ZIRCON","GLASS"],"analyte":["SiO2","CaO"],"analysisMethod":["EMP","SIMS"]}

Data Search Examples

  • Electron microprobe analyses on Anorthosites (including rocks and minerals)
    • http:// api.moondb.org/data/{"mission":[],"landmark":[],"specimenType":["Anorthosite"],"samplingTechnique":[],"analyzedMaterial":[],"analyte":[],"analysisMethod":["EMP"]}
  • Water analyses on lunar soil samples
    • http:// api.moondb.org/data/{"mission":[],"landmark":[],"specimenType":["SOIL"],"samplingTechnique":[],"analyzedMaterial":[],"analyte":["H2O"],"analysisMethod":[]}
  • Th measurements on all Rock samples
    • http:// api.moondb.org/data/{"mission":[],"landmark":[],"specimenType":["ROCK"],"samplingTechnique":[],"analyzedMaterial":[],"analyte":["Th"],"analysisMethod":[]}
  • Basalts with Ti data (both Ti and TiO2) from all missions with any method
    • http:// api.moondb.org/data/{"mission":[],"landmark":[],"specimenType":["Basalt"],"samplingTechnique":[],"analyzedMaterial":[],"analyte":["Ti","TiO2"],"analysisMethod":[]}
